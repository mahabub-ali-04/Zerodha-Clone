# Zerodha Clone (Full Stack)

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Responsive UI
- REST APIs for holdings/positions/orders
- Backend + frontend structured as a monorepo

## Folder Structure
- /frontend
- /dashboard
- /backend

## How to Run
### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

### Dashboard
cd dashboard
npm install
npm start
